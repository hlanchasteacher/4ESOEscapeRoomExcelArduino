🔐 Room 6 — LEDs & Digital Output

Q16. An LED has two legs. The longer leg is the anode. Where does the anode connect in a basic circuit?
✗ LEDs are POLARISED — they only work one way! Anode (+) → resistor → Arduino pin. Cathode (−) → GND.
✓ Anode (+, longer leg) connects via a resistor to the Arduino pin. Cathode (−, shorter leg) to GND. Polarity matters!
LED = Light Emitting DIODE. Diodes only let current flow ONE way. Anode (+) = longer leg → positive side. Cathode (−) = shorter leg, flat edge → GND.
Q17. Why do we always put a resistor in series with an LED in an Arduino circuit?
✗ LEDs have very low resistance — without a current-limiting resistor, too much current flows and they burn out instantly.
✓ Current limiting! Ohm's Law: V = I × R. A 220Ω resistor with a 5V Arduino pin gives about 20mA — the safe operating current for most LEDs.
LEDs are very low resistance. Without a resistor, the current is V/R = 5/~0 = HUGE → instant burnout. Resistor = current limiter = LED protector.
Q18. Which Arduino function turns a digital pin ON (5V / HIGH)?
✗ ledOn, pinWrite, setPin don't exist in Arduino! The correct function is digitalWrite(pin, value).
✓ digitalWrite(13, HIGH)! Sets pin 13 to 5V. Use digitalWrite(13, LOW) to turn it off. Remember: you also need pinMode(13, OUTPUT) in setup()!
Arduino core functions: digitalWrite(pin, HIGH/LOW), digitalRead(pin), analogRead(pin), analogWrite(pin, 0-255). Don't invent function names!

📡 HC-SR04 Ultrasonic Sensor — How it works
HC-SR04 Pins: Arduino Uno ┌──────────┐ ┌──────────────┐ │ VCC ├──────┤ 5V │ │ TRIG ├──────┤ Pin 9 │ ← sends 10µs pulse │ ECHO ├──────┤ Pin 10 │ ← receives reflected pulse │ GND ├──────┤ GND │ └──────────┘ └──────────────┘ How it measures distance: 1. Q22. Looking at the diagram: which pin do you send a 10µs pulse to in order to trigger a measurement?

✓ TRIG! Send a 10µs HIGH pulse to Pin 9 (TRIG) to trigger the sensor. Then read Pin 10 (ECHO) with pulseIn() to measure the return time.
TRIG = Trigger (you send the signal). ECHO = Echo (you receive the reflected signal). VCC = power, GND = ground.
Q23. The Arduino function used to measure how long the ECHO pin stays HIGH is:

✓ pulseIn(ECHO, HIGH)! Returns the duration in microseconds. Use it to calculate distance using the speed of sound.
pulseIn(pin, HIGH) = measures how many microseconds the pin stays HIGH. Perfect for timing echo pulses. digitalRead only tells you if it's HIGH or LOW — not for how long.
